Menu Panel by William Chang

The menu panel component can take a nested list items up to 3-levels deep. The nested list doesn't need extra attributes like ID's and CSS classes.

Supported browsers: Microsoft Internet Explorer 6+, Mozilla Firefox 3+, Apple Safari 3+, Google Chrome 1+

Features

  • Easy and simple to apply to multiple lists.
  • Customizable through settings and CSS.
  • Attach callback functions to load event.

API and Options

You can pass options as key/value object to the component's function.

Key Type Default Value Description
strRegionId string region_header The container that surrounds markup structure of all nested lists. Requires a CSS id.
strPanelsSelector string ul The container that surrounds the list items markup structure. Requires a CSS selector.
strItemsSelector string li The items of the list markup structure. Requires a CSS selector.
strHasChildrenSelector string > ul The nested list inside each item markup structure. Requires a CSS selector.
intItemsGroup integer 3 The third level of the nested list items will be grouped or limited to each column.
strLevel1MenuPanelCss string menu-level1-panel The first level of the nested list. Requires a CSS class name.
strLevel1MenuItemCss string menu-level1-item The first level of the nested list. Requires a CSS class name.
strLevel1MenuItemFirstCss string menu-level1-item-first The first level of the nested list. Requires a CSS class name.
strLevel1MenuItemLastCss string menu-level1-item-last The first level of the nested list. Requires a CSS class name.
strLevel1MenuLinkCss string menu-level1-link The first level of the nested list. Requires a CSS class name.
strLevel2MenuPanelCss string menu-level2-panel The second level of the nested list. Requires a CSS class name.
strLevel2MenuItemCss string menu-level2-item The second level of the nested list. Requires a CSS class name.
strLevel2MenuItemFirstCss string menu-level2-item-first The second level of the nested list. Requires a CSS class name.
strLevel2MenuItemLastCss string menu-level2-item-last The second level of the nested list. Requires a CSS class name.
strLevel2MenuItemHasChildrenCss string menu-level2-item-haschildren The second level of the nested list. Requires a CSS class name.
strLevel2MenuItemHasLonersCss string menu-level2-item-hasloners The second level of the nested list. Requires a CSS class name.
strSelectedMenuItemCss string menu-item-selected The second level of the nested list. Requires a CSS class name.
fnLoad function null Callback function after finishing setting up the menu panel.

Usage

1. Include required dependencies.


<script src="../js/jquery.js" type="text/javascript"></script>
<script src="../js/diehard_menupanel.js" type="text/javascript"></script>

2. Add required markup structure using HTML.


<div id="region_menu">
	<ul>
		<li>
			<a href="#">Menu Item 1</a>
			<ul>
				<li>
					<a href="#">Menu Item 1.1</a>
					<ul>
						<li><a href="#">Menu Item 1.1.1</a></li>
						<li><a href="#">Menu Item 1.1.2</a></li>
						<li><a href="#">Menu Item 1.1.3</a></li>
						<li><a href="#">Menu Item 1.1.4</a></li>
						<li><a href="#">Menu Item 1.1.5</a></li>
						<li><a href="#">Menu Item 1.1.6</a></li>
					</ul>
				</li>
				<li>
					<a href="#">Menu Item 1.2</a>
					<ul>
						<li><a href="#">Menu Item 1.2.1</a></li>
						<li><a href="#">Menu Item 1.2.2</a></li>
						<li><a href="#">Menu Item 1.2.3</a></li>
						<li><a href="#">Menu Item 1.2.4</a></li>
					</ul>
				</li>
				<li><a href="#">Menu Item 1.3</a></li>
				<li><a href="#">Menu Item 1.4</a></li>
				<li><a href="#">Menu Item 1.5</a></li>
			</ul>
		</li>
		<li>
			<a href="#">Menu Item 2</a>
		</li>
	</ul>
</div>

3. Add required style using CSS.


/* Navigation */
#region_menu {position:relative;top:0;right:0;width:99%;height:40px;margin-bottom:10px;font-size:80%;border:solid 1px green;}
#region_menu ul {margin:0;padding:0;list-style-type:none;}
#region_menu li {margin:0;padding:0;}
#region_menu a {display:block;}
/* First Level */
#region_menu .menu-level1-item {float:left;width:auto;margin:0 0 0 16px;padding:0 0 4px 0;border:solid 1px blue;}
#region_menu .menu-level1-item a {padding:6px 6px;}
/* Second Level */
#region_menu .menu-level2-panel {display:none;position:absolute;top:26px;left:0;padding:8px;border:solid 1px red;background-color:#eee;}
#region_menu .menu-level2-panel a {padding:0 0;font-weight:bold;}
#region_menu .menu-level1-item1 .menu-level2-panel {left:0px;}
#region_menu .menu-level1-item2 .menu-level2-panel {left:100px;}
#region_menu .menu-level1-item3 .menu-level2-panel {left:200px;}
#region_menu .menu-level1-item4 .menu-level2-panel {left:280px;}
#region_menu .menu-level1-item5 .menu-level2-panel {left:430px;}
#region_menu .menu-level1-item6 .menu-level2-panel {left:450px;}
#region_menu .menu-level2-item {float:left;width:auto;padding-left:6px;border-left:solid 1px cyan;}
#region_menu .menu-level2-item-first {padding-left:0;border-left:0;}
#region_menu .menu-level2-item-haschildren {}
#region_menu .menu-level2-item-hasloners {min-width:100px;min-height:60px;}
#region_menu .menu-level2-item-hasloners a {margin:0 4px 6px 0;}
/* Third Level */
#region_menu .menu-level2-item ul li {padding-right:6px;}
#region_menu .menu-level2-item-first ul li {}
#region_menu .menu-level2-item-haschildren ul li {float:left;width:auto;}
#region_menu .menu-level2-item-haschildren ul li a {font-weight:normal;}

4. Add and run the code using JavaScript.


$('#region_menu ul:eq(0)').diehard_menupanel();
